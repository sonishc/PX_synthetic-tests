# PX_synthetic-tests
Use for production test case

## How to run local

```
git clone https://github.com/sonishc/PX_synthetic-tests.git
```
```
cd PX_synthetic-tests
```
Create ENV variable:
 ```
  export USER_EMAIL='your_email'
  export USER_PASSWORD='your_password'
 ```
```bash
ruby task_runner.rb
```
